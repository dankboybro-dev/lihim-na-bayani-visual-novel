# Lihim na Bayani (Starter)

A minimal, self-contained **command-panel narrative game** starter you can run in any modern browser. 
It includes a main menu, chapter select, and three placeholder chapters you can replace with your own content.

## How to run
1. Download and unzip this project.
2. Open `index.html` in your browser (double-click).
   - If your browser blocks local scripts, use a simple local server instead:
     - Python: `python -m http.server 8000` then visit http://localhost:8000
3. Edit `app.js` to change chapters, lines, and choices.
4. Edit `styles.css` to customize the UI.

## Files
- `index.html` – Shell page
- `styles.css` – Minimal UI styles
- `app.js` – Narrative logic and placeholder content

Enjoy building your *Lihim na Bayani* story!
